Core Capabilities

Enterprise-grade torrent discovery infrastructure

Unified Media Search

Integrated with The Movie Database (TMDB) to provide comprehensive search functionality across millions of movies and television series.

Multi-Source Aggregation

Consolidates torrent data from multiple indexing services including Torrentio, Comet, StremThru, and PirateBay+ into a single interface.

Granular Episode Control

Full support for television content with season and episode-level navigation, enabling precise selection of individual episodes with metadata.

Optimized Performance

Built with Redis caching layer and intelligent request handling to deliver consistent response times under high concurrency.

Commands

Simple slash commands to get started

/credits View bot credits, data sources, and project information.
